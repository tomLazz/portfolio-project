8 Puzzle Solver

C, Python

Created a large dataset of 8 puzzle game states and corresponding moves with an implementation of A* algorithim created using C. This dataset was used to train a Convolutional Neural Network, created with Pytorch, to predict the best move for a given board with ~85% accuracy. This predictive model was used as a hurestic in an informed search algorithm, ultimately creating a much less computationally costly approach for solving the 8 puzzle than traditional solutions.

These functions encompass converting formatted strings to regular expressions encoded as recursive variant types, verifying string acceptance in a set via a nondeterministic finite automaton (NFA) record, transforming regular expressions into NFA representations, and further converting NFAs to DFAs for greater efficiency.
